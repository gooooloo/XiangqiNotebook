# Changelog

All notable changes to 象棋笔记本 (XiangqiNotebook) will be documented in this file.

---

## 1.0.5

- 新增皮卡鱼快速应招功能，一键获取引擎推荐着法并自动走出
- 新增皮卡鱼快速估分和快估本局功能，3秒限时快速评估局面
- 原皮卡鱼查分改名为皮卡鱼深度评分，与快速估分独立显示
- 新增实战列表功能，显示当前局面相关的实战对局，支持右键筛选
- 新增实战列表显示/隐藏开关，关闭时提升切换局面性能
- 新增PGN导出功能，支持导出所有变着或仅导出当前棋局
- 新增生成详细局面文本面板，支持预览和一键拷贝
- 状态栏新增FEN显示和拷贝功能
- 状态栏新增走棋方提示
- 棋局浏览器记住折叠状态和选中项
- 招法列表中标记已加入复习库的步骤
- 复习模式下默认隐藏路径和下一步着法，更好测试记忆
- 修复复习模式下键盘快捷键失效问题
- 修复实战列表缓存导致切换局面变慢的问题
- 修复重新统计实战统计功能
- 增大局面文本对话框尺寸，解决内容显示不全问题

---

## 1.0.4

- 新增复习模式，基于 SM-2 间隔重复算法自动安排复习计划
- 新增复习库管理，支持将局面加入或移出复习库
- 新增下步变招列表，与本步变招并排显示；点击下步变招仅切换分支，不自动前进
- 精简界面按钮布局，重组菜单栏结构
- 修复练习模式下路径信息仍然可见的问题

---

## 1.0.3

- macOS 新增原生菜单栏，所有操作均可通过菜单执行，并显示快捷键
- macOS 新增可自定义工具栏
- macOS 新增 HTTP 服务用于接收 PGN 棋谱导入
- 新增一键练习先手/后手开局按钮
- 参考棋谱窗口改为累积显示，不再每次替换
- 棋局标题中的"vs"替换为胜/和/负结果显示
- 启用"只筛选特定棋局"时显示对应棋局标题
- Pikafish 评分显示增加好棋/坏棋颜色标识
- 修复棋局统计行布局对齐问题
- 修复执黑时棋局标题显示错误结果的问题
- 修复搜索当前着法时的匹配逻辑问题

---

## 1.0.2

- 新增 PGN 棋谱批量导入功能，支持导入他人对局
- 实战棋局按日期倒序排列，日期显示增加具体时间
- 修复删除棋局时的崩溃问题
- 修复备份恢复时的崩溃问题
- 优化棋书目录树展开/折叠按钮的点击区域
- 修复应用图标在部分尺寸下显示不清晰的问题
